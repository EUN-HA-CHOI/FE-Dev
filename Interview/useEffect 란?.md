## useEffect 란?

`useEffect` 는 리액트의 훅(Hooks) 중 하나, 함수형 컴포넌트에서 부작용을 처리하기 위해 사용된다.  
부작용은 컴포넌트의 렌더링과 관련이 없는 작업들을 말하며, 주로 데이터 가져오기, 구독 설정, 이벤트 처리 등이 해당된다.  

`useEffect` 훅을 사용하면 컴포넌트가 렌더링 될 때마다 특정 동작을 수행하거나 컴포넌트의 상태나 속성이 변경될 때마다  
특정 동작을 수행할 수 있다.  

```javascript
useEffect(() => {
  // 부작용을 수행하는 코드
  return () => {
    // 정리(clean-up) 함수
  };
}, [dependency]);

```
